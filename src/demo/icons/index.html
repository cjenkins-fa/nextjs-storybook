Icons

Our icon library has 3 distinct sizes: small (20px x 20px), medium (40px x 40px) and large (130px x 91px). These come in SVG format and are added as a background image. To use them, simply put a <span> element with the following classes: class="icon icon--[icon-name]".

For example, this icon is produced by the following code:

<span class="icon icon--calculator"></span>

Nav/System icons

While the use of these are meant for navigation, these can be used anywhere without discrimination. Please note that the icons are not perfectly aligned to the center for visual alignment purposes when used.

Custom colored icons

To get custom colors on the icons you must use our SASS library and call our get-icon() function from our icon library and pass the name of the icon you would like to get and pass on up to two colors if the icon supports it. If id does not, it will only use the first color and the second one will be ommited. If you pass only one color, it will only use one color for the icon.

For example, we can add a class called custom-calculator to the SASS of the page. To get the custom colors, we just need to call the get-icon function on the background-image property like this: get-icon('calculator', #000, #fff).

Here is a custom calculator that supports 2 colors with the code needed to produce it:

// index.html


<span class="icon custom-calculator"></span>

// style.scss


@use '@firstam_enterprise_web/fawc/src/stylesheets/base/svg-icons' as *;

.custom-calculator {
  background-image: get-icon('calculator', #141d3a, #4eade1);
}

Social Icons

These are the social icons that we have available. They are all the same size and can be used in the same way as the system/small icons on the page as you see fit. Though, these come with their own colors and cannot be customized. If needed, other copies of the icons would be added with just 2 colors (black and white) that can be swapped. For now these are the only social icons in the design system.

Here is the code to produce all of these icons in the same order:


<span class="icon small icon--linkedin"></span>
<span class="icon small icon--facebook"></span>
<span class="icon small icon--youtube"></span>
<span class="icon small icon--google"></span>
<span class="icon small icon--x"></span>

Custom external icons

If you want to include new SVG icons that are not in Core but rather a new icon needed for a new site, you can use the get-icon() function and provide as the last argument OR specify the target argument a SASS map containing the SVG code of your custom icon (this is, of course, after importing the core.scss file with the @use rule at the top of your file to make sure the funciton is avaiable in your scope). If you want to be able to change the colors, make sure to swap the color values to color1 and color2 (e.g. stroke="color1" or fill="color1"). Here is an example of a SASS map whichcontains the code of a custom icon: 


$custom-icons: (
  residential-custom:
    '<svg xmlns="http://www.w3.org/2000/svg" width="81" height="64" fill="none"><path stroke="color1" stroke-linecap="round" stroke-linejoin="round" stroke-width="2.3" d="M40.502 61.94h-7.4v-9.4c0-2 1.6-3.8 3.6-3.8s3.6 1.6 3.6 3.8v9.4h.2ZM40.502 39.34h-7.4v-9.4c0-2 1.6-3.8 3.6-3.8s3.6 1.6 3.6 3.8v9.4h.2ZM64.3 61.94h-7.4v-9.4c0-2 1.6-3.8 3.6-3.8s3.6 1.6 3.6 3.8v9.4h.2ZM64.3 39.34h-7.4v-9.4c0-2 1.6-3.8 3.6-3.8s3.6 1.6 3.6 3.8v9.4h.2ZM11.102 46.94v15M20.3 44.14c0 5.2-4.2 9.4-9.2 9.4-5 0-9.2-4.2-9.2-9.4s4.2-14 9.2-14c5 0 9.2 8.8 9.2 14ZM79.101 20.74l-14.6-18.8m0 0h-29.4l-14.6 18.8h31.2l12.8-18.8ZM47.9 26.34v35.6M27.7 26.34v35.6"/><path stroke="color2" stroke-linecap="round" stroke-linejoin="round" stroke-width="2.3" d="M3.7 61.94h69.8v-41.2"/></svg>',
);

Notice how the color values were replaced with color1 and color2. This is so that you can change the colors of the icon with the function as well as needed. The other piece of code you need in order to have this icon available as if it were part of the icon library is to create your custom class that gets this icon. Please notice the order of arguments passed to the function.


.icon--residential-custom {
  background-image: get-icon('residential-custom', $icons-map: $custom-icons);
}
// Or
.icon--residential-custom {
  background-image: get-icon('residential-custom', $vivid, $welcome, $custom-icons);
}

After the above code is in your site specific SASS file, you can now call your custom icon class like this:


<span class="icon large icon--residential-custom"></span>
