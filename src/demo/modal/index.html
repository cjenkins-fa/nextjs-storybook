Modal Windows

Modal windows are a great way to display content without navigating away from the current page. They can be used for a variety of purposes, such as displaying forms, images, videos, or other content.

The trigger button or a should contain a data-target attribute that matches the id of the modal window, a data-title attribute that will be used as the title of the modal window, and a data-toggle="modal" attribute that will be use to add the event listener.

Here are a few examples of modal windows:

Basic Modal

This is a basic modal window with a close button and a title.

Contact Us 


  
  <button type="button" class="button button__primary welcome" data-toggle="modal" data-title="Demo Modal"
  data-target="demo-modal">
      Demo Modal
  </button>

  
  <div id="demo-modal" class="modal" role="dialog" aria-labelledby="modal window">
    <div class="modal__dialog">
      <div class="modal__content">
        <div class="modal__header">
          <h4 class="modal__title" data-title="Contact Us">Contact Us</h4>
          <button type="button" class="close">
            <svg xmlns="http://www.w3.org/2000/svg" width="20" height="20" viewBox="0 0 20 20">
              <path d="M1.63605 18.3647L18.364 1.63682M1.63605 1.63672L18.364 18.3646" stroke="white" stroke-width="2" stroke-linecap="round" />
            </svg>
            <span class="sr-only">Close</span>
          </button>
        </div>
        <div class="modal__body">
          <script>
            hbspt.forms.create({
              portalId: "17501",
              // Dev Form -----
              formId: "fa649f42-f1c5-4cb5-8ba9-012b077b5c79",
              onFormReady: function ($form) {
                window.hubspotForms.init();
              },
              onBeforeFormSubmit: function ($form, submissionValues) {
                $($form).find('input[name="target_contact"]').val($('.modal-title').text());
              }
            });
          </script>
        </div>
      </div>
    </div>
  </div>
